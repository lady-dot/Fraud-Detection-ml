# Fraud-Detection-ml
Machine Learning model for fraud detection

## Overview
This project detects fraudulent financial transactions using machine learning techniques.

## Dataset
Public credit card fraud dataset from Kaggle.

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Methodology
1. Data preprocessing and feature scaling
2. Model training and testing
3. Performance evaluation
4. Web app deployment using Streamlit

## Results
The model achieved 94.6% accuracy in detecting anomalous transactions.

## How to Run
1. Install requirements
2. Run `streamlit run app.py`

- Improve recall on minority class
